# Manager Cosmetice

## Description
Manager Cosmetice is a comprehensive application designed for managing various cosmetic products, enabling users to track their inventory, view product details, and stay updated with the current trends in the cosmetic industry.

## Purpose
The primary purpose of this project is to provide an easy-to-use platform for individuals and businesses to manage their cosmetic products efficiently. This application aims to simplify inventory management and enhance user experience with its intuitive interface.

## Technology Stack
- **Backend:** Java (57.2%)
- **Frontend:** HTML (42.8%)

## Key Features
- **Product Management:** Add, update, and delete cosmetic products easily
- **Inventory Tracking:** Keep track of product stock levels
- **Product Details:** View comprehensive information about each cosmetic product
- **User-Friendly Interface:** Navigate through the application with ease

## Project Structure
The project is organized with:
- Java backend components for business logic and database operations
- HTML frontend components for user interface

## How to Use
1. Clone the repository
2. Set up the necessary dependencies
3. Configure the database connection
4. Run the application
5. Access the interface through your web browser
